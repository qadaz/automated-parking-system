# Automated Parking System 🚗

A Python-based automated parking management system designed to simulate real-world parking operations. This project demonstrates programming logic, data handling, and basic system automation concepts.

---

## 📌 Project Overview

The system allows users to:
- Time simulation 
- Park vehicles
- Remove vehicles
- Track available parking spaces
- Calculate parking fees 
- Display parking status

This project was developed to strengthen Python programming skills and apply logical problem-solving techniques to a practical real-world scenario.

---

## 🛠️ Technologies Used

- Python 3
- Object-Oriented Programming 
- Basic file handling 

---

## 💡 Skills Demonstrated

- Python programming
- Control structures (loops, conditions)
- Functions and modular design
- Data structures (lists, dictionaries)
- Problem-solving and logical thinking
